# Dashboard

The dashboard is your control panel for managing JSON files.

## My JSONs

Lists all uploaded files with:

- **Filename** — displayed with `.json` extension
- **Created** — upload date
- **URL button** — click to copy the public API URL to clipboard

Clicking the copy icon copies the full URL:

```
https://json-server.shahriyar.dev/<username>/<filename>
```

## Upload

Drag-and-drop file upload with:

- File validation (`.json` only)
- Filename editing
- Real-time validation feedback
- URL preview before upload

On success, the URL is automatically copied to your clipboard and you're redirected to My JSONs.
