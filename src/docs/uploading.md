# Uploading

## How to Upload

1. Sign in with GitHub
2. Navigate to the Upload page
3. Drag and drop a `.json` file onto the dropzone, or click to browse
4. Edit the filename if needed
5. Click Upload

On success, the public URL is automatically copied to your clipboard and you're redirected to My JSONs.

## Rules

- **Filename**: letters, numbers, dashes, and underscores only. No spaces.
- **File type**: must be valid `.json`
- **Uniqueness**: filenames are unique per user. Uploading with an existing name overwrites the old file.
- **Size**: governed by server payload limits.

## Managing Files

Go to **My JSONs** in the dashboard to see all your uploaded files. Each entry shows the filename, creation date, and a copy-URL button.
